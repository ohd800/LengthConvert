Conversion de longueurs

Résultat :

Informations sur le kilomètre :

Définition du kilomètre : Le kilomètre (symbole : km) est l'unité de base de longueur dans le système international d'unités (SI). Il est défini comme 1000 mètres.

Exemples :

Utilisation standard :
• La distance entre deux villes peut être mesurée en kilomètres.
• La longueur d'une course de marathon est d'environ 42,195 kilomètres.

Mesures quotidiennes :
• Estimer la longueur d'un sentier de randonnée en kilomètres.
• Représenter la distance entre deux endroits sur une carte en kilomètres.

Faits :

Préfixes du système métrique :
• Le kilomètre est basé sur l'unité métrique mètre, avec le préfixe "kilo" indiquant qu'il est équivalent à mille mètres.

Système décimal :
• Les préfixes du système métrique, y compris "kilo", suivent le système décimal, facilitant les conversions entre différentes unités.

Utilisation mondiale :
• Le kilomètre est largement utilisé dans le monde entier, en particulier dans les pays adoptant le système métrique.

Questions d'exercice :

Conversion :
• Convertir 5000 mètres en kilomètres.
• Représenter 2,5 kilomètres en mètres.

Application :
• Si une voiture se déplace à une vitesse de 80 kilomètres par heure, quelle distance parcourra-t-elle en 3 heures ?
• La longueur d'un terrain rectangulaire est de 1500 mètres, et sa largeur est de 800 mètres. Quelle est sa superficie en kilomètres carrés ?

Réponses :

Conversion :
• 5000 mètres = 5 kilomètres
• 2,5 kilomètres = 2500 mètres

Application :
• Distance = Vitesse × Temps
• Distance = 80 kilomètres/heure × 3 heures = 240 kilomètres
• Superficie = Longueur × Largeur
• Superficie = (1500 mètres × 800 mètres) / (1000 mètres/kilomètre)² = 1,2 kilomètre carré
